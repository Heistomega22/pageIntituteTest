# Sistema web del instituto12 para el instituto15
Sistema web desarrollado para la gestion de alumnos, profesores y cursos del instituto ifts15.




## Lenguajes utilizados
- HTML
- CSS
- JavaScript
- PHP
- SQL

## Tecnologias utilizadas
- Node.js
- Express.js
- MySQL

## Objetivo del proyecto
- Gestionar alumnos, profesores y cursos del instituto15.
- Permitir a los profesores tener un espacio por cada curso para subir materiales y tareas, en dichas tareas los alumnos pueden subir sus trabajos.
- Permitir a los profesores descargar, editar, dejar comentarios y calificar los trabajos subidos por los alumnos.
- Permitir a los alumnos ver sus cursos, materiales y tareas, y subir sus trabajos.
- Permitir a los administradores gestionar usuarios, cursos y ver reportes.
- Permitir a los alumnos ver sus notas y reportes de sus cursos.
- Permitir a los profesores ver reportes de sus cursos y alumnos.
